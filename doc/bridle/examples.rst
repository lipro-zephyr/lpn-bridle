.. _examples:

Applications and samples
########################

The `Zephyr Project`_ contains a variety of application samples and demos.
Documentation for those is available in the
:zephyr:code-sample-category:`samples` section.

|BRIDLE| provides additional examples that specifically target |TIAC| devices
and show how to implement typical use cases with our libraries and drivers.
Samples showcase a single feature or library, while applications include a
variety of libraries to implement a specific use case.
